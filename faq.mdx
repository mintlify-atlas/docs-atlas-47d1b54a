title: "FAQ"
description: "Answers to the most common questions about Anikin."
---

<AccordionGroup>
  <Accordion title="Does Anikin require an account?">
    No. You can browse, stream, and download anime and manga without creating an account. Signing in with AniList is optional — it enables online sync of your favourites list and watched progress across devices.
  </Accordion>

  <Accordion title="Is Anikin free?">
    Yes. Anikin is free and open-source. There are no subscriptions, in-app purchases, or ads. The source code is available on [GitHub](https://github.com/jerry08/Anikin).
  </Accordion>

  <Accordion title="What platforms does Anikin support?">
    Anikin currently supports:

    - **Android** — Android 5.0 (API 21) and later
    - **Windows** — Windows 10 version 1803 (build 17763) and later

    Download the APK for Android or the ZIP package for Windows from the [GitHub Releases](https://github.com/jerry08/Anikin/releases) page.
  </Accordion>

  <Accordion title="Why does a video not play?">
    The most common cause is that the current provider is temporarily down or does not have the episode available. Try the following:

    1. Open the episode page and tap the provider selector.
    2. Switch to a different provider.
    3. If the problem persists, check your internet connection and try again.

    See [Troubleshooting](/troubleshooting) for more detailed steps.
  </Accordion>

  <Accordion title="How do I update Anikin?">
    Anikin checks for updates automatically on startup. When a new version is available, an update sheet appears with release notes and a download link.

    You can also update manually at any time:

    - **Android** — download the latest APK from [GitHub Releases](https://github.com/jerry08/Anikin/releases) and install it over your existing installation.
    - **Windows** — download the latest ZIP from [GitHub Releases](https://github.com/jerry08/Anikin/releases), extract it, and replace the existing files.

    <Tip>If automatic update checks are not appearing, make sure **Always check for updates** is enabled in **Settings**.</Tip>
  </Accordion>

  <Accordion title="How do I report a bug or request a feature?">
    You can report bugs and request features in two places:

    - **GitHub Issues** — [github.com/jerry08/Anikin/issues](https://github.com/jerry08/Anikin/issues). Use the issue templates to provide the right information.
    - **Discord** — join the community server at [discord.com/invite/U7XweVubJN](https://discord.com/invite/U7XweVubJN) and post in the relevant channel.
  </Accordion>

  <Accordion title="Why can't I find a specific anime?">
    Each provider has its own catalogue. If an anime does not appear in the search results, try the following:

    1. Switch to a different provider using the provider selector on the episode page.
    2. Use the manual provider search — when no match is found automatically, a search sheet opens so you can search by title within the selected provider.
    3. Try alternate title spellings (romaji, English, or native).
  </Accordion>

  <Accordion title="Does Anikin support dubbed anime?">
    Yes. On the episode page, toggle the **Dub** option to switch between subtitled and dubbed versions. Availability depends on the selected provider.
  </Accordion>

  <Accordion title="How do I sync my watch history across devices?">
    Log in with your AniList account. Once authenticated, your favourites list and watched progress sync with AniList automatically. See [AniList login](/account/anilist-login) and [Sync & progress](/account/sync-and-progress) for setup instructions.
  </Accordion>

  <Accordion title="Where are downloaded files saved?">
    Downloaded files are saved to platform-specific locations:

    - **Android** — the public **Downloads** folder on your device (accessible through your file manager or the Downloads app).
    - **Windows** — your user **Videos** folder (`%USERPROFILE%\Videos`).
  </Accordion>
</AccordionGroup>
