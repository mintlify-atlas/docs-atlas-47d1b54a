---
title: "Installation"
description: "Download and install Anikin on Android or Windows."
---

Anikin is distributed as a standalone package — no app store required. Pick your platform below and follow the steps to get set up.

<Tip>
  Always download the latest version from the [GitHub Releases page](https://github.com/jerry08/Anikin/releases) to get the newest features and bug fixes.
</Tip>

<Tabs>
  <Tab title="Android">
    **Supported:** Android 5.0 (API level 21) and higher

    <Warning>
      Android blocks installation of apps from outside the Play Store by default. You need to enable **Install unknown apps** (sometimes called "Install from unknown sources") in your device settings before installing the APK. The exact location of this setting varies by Android version and device manufacturer.
    </Warning>

    <Steps>
      <Step title="Download the APK">
        Go to the [Anikin Releases page](https://github.com/jerry08/Anikin/releases) on GitHub and download the latest `.apk` file (for example, `Anikin-v2.0.2.apk`).
      </Step>
      <Step title="Allow installation from unknown sources">
        Open your device **Settings**, then navigate to **Security** (or **Privacy** on some devices) and enable **Install unknown apps** for your browser or file manager. On Android 8.0 and later, you may be prompted to grant this permission when you open the APK file.
      </Step>
      <Step title="Open the APK">
        Open the downloaded `.apk` file using your file manager or the notification in your browser. Tap **Install** when prompted.
      </Step>
      <Step title="Grant permissions">
        If the app requests storage permissions, tap **Allow**. This is required for downloading episodes and saving app data.
      </Step>
      <Step title="Launch Anikin">
        Tap **Open** after installation completes, or find **Anikin** in your app drawer.
      </Step>
    </Steps>
  </Tab>
  <Tab title="Windows">
    **Supported:** Windows 10 version 17763.0 (October 2018 Update) and higher

    <Steps>
      <Step title="Download the ZIP">
        Go to the [Anikin Releases page](https://github.com/jerry08/Anikin/releases) on GitHub and download the latest Windows archive (for example, `Anikin-windows-v2.0.2.zip`).
      </Step>
      <Step title="Extract the archive">
        Right-click the downloaded `.zip` file and select **Extract All**, then choose a destination folder. You can place the extracted folder anywhere — a location like `C:\Users\YourName\Apps\Anikin` works well.
      </Step>
      <Step title="Run the app">
        Open the extracted folder and double-click **Anikin.exe** to launch the app. No installation wizard is required.
      </Step>
    </Steps>

    <Note>
      Windows may show a SmartScreen warning the first time you run the executable because the app is not distributed through the Microsoft Store. Click **More info**, then **Run anyway** to proceed.
    </Note>
  </Tab>
</Tabs>
