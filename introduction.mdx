Anikin is a multi-platform application that lets you stream anime and read manga from multiple content providers. It parses data directly from provider websites, stripping ads so you get a clean playback experience. You don't need to create an account to start watching or reading.

## What makes Anikin different

- **No account required.** You can browse, search, and stream immediately after installing the app.
- **Ad-free.** Anikin parses content directly from providers and strips ads from video playback.
- **Multi-platform.** A single codebase targets Android 5.0+ and Windows 10, so the experience is consistent across devices.
- **Open source.** The full source is available on [GitHub](https://github.com/jerry08/Anikin) and contributions are welcome.

## Feature overview

| Feature | Details |
|---|---|
| Search | Find anime and manga by title |
| Recently added | Browse titles with new episodes in the last 7 days |
| Popular | Browse all-time popular anime and manga |
| Movies | Filter for anime films |
| Favourites | Mark titles and episodes as favourites |
| Auto quality | Automatically selects video quality for playback |
| Ads-free playback | No ads in the video player |
| AniList sync | Sync favourites and watch progress via AniList login |
